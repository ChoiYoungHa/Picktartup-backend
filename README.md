# 👀 Intro
![image](https://github.com/user-attachments/assets/824668ed-15c7-419a-b61c-cd2e6d2c8abe)
- 스타트업은 자금난에 시달리고 있지만 개인이 비상장 스타트업에게 투자할 방법이 거의없다.


![image](https://github.com/user-attachments/assets/590fcbed-4507-4d25-95f5-0c67194eb737)
- 스타트업은 VC, 은행 이외에 투자자를 찾기 어려운 문제가 있음
- 개인투자자가 스타트업에게 투자하려면 높은 진입장벽과 최소투자 금액이 너무 높은 문제가 발생


![image](https://github.com/user-attachments/assets/7dc8a8c1-3ab7-41d6-9bc5-a42fbfddf654)
- 따라서 블록체인 기술을 사용하여 투명성, 신뢰성, 접근성, 편의성까지 모두 충족시키고 스타트업의 자금난과 개인 투자자의 투자처 확대를 도와줄 "픽타트업" 서비스를 기획함


<br>

# 1️⃣ 프로젝트 소개

### 🔔 핵심 포인트
1. 접근성 : **블록체인** 기반 토큰증권으로 개인투자자의 스타트업 투자 진입장벽 완화
2. 신뢰성 : 블록체인을 통한 **투명한 거래 기록** 보관 및 관리
3. 효율성 : 지분의 **토큰화**를 통한 소액투자 가능

### 🎈 서비스
* 회원 관리 시스템
  * 자체 회원가입/로그인 기능
  * 마이페이지를 통한 회원정보 및 지갑주소 관리

* 블록체인 지갑 시스템
  * 개인별 지갑 생성 및 관리
  * 실시간 잔고 조회 및 토큰 거래 기능

* 토큰 관리 시스템
  * 결제 완료 시 자동 토큰 발행
  * 블록체인 네트워크 기반 실시간 잔고 업데이트

* 투자 계약 관리
  * 자동화된 계약서 생성 및 관리
  * 목표 미달성 시 자동 환불 시스템
  * 계약 진행상황 실시간 모니터링

* 코인 거래 시스템
  * 편리한 코인 구매 및 현금화 기능
  * 상세한 거래내역 조회 서비스

* 스타트업 정보 플랫폼
  * 키워드 기반 스타트업 검색
  * SSI(Start-up Success Index) 기반 투자 정보 제공
  * 상세한 기업 정보 열람 서비스

* 관리자 시스템
  * 실시간 모니터링 시스템

스타트업의 재무현황, MAU, 고용인원, 자체적인 기준으로 스타트업의 성장지수를 보여주는 Smart Startup Index 지표(팀, 제품, 잠재성, 성과) 등 사용자의 투자에 도움될 수 있도록 지표를 제공합니다.

**투자실행**

투자할 토큰을 입력 후 계약서를 읽고 서명하면 투자가 실행됩니다. 이후 스타트업이 지정한 금액만큼 모금이 완료되면 투자는 실행되고, 모금액이 모이지 않으면 토큰은 반환됩니다.
